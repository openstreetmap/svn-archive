The attached script, shape_to_osm-Tiger.py provides a quick-and-easy way to convert US Census Beaureau Tiger shapefiles to .osm format.  It is useful for comparing the latest Tiger data to the older data already in OSM.  However, a more sophisticated converter would be desireable in preparing data for import.

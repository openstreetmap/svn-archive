Hilfe

Start

  • Wähle einen Bereich aus, für den die Karte erstellt werden soll.
  • Wähle einen Ordner aus, in den die Dateien gespeichert werden sollen.
  • Lege einen Filter an
  • gib dem Filter
    • einen Namen(eindeutig)
    • einen Dateinamen (zb Filtername+".txt")
  • weise ein Icon zu
    • einen Dateinamen (zb Filtername+".png")
    • eine Größe (zb "20,20")
    • Einen Iconversatz (um das Icon mittig zu platzieren, wähle die Hälfte der Größe: "10,10")

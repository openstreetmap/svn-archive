Freemap 3D

This is a demo application to render OpenStreetMap data in 3D using WebGL. This is a very early version, so it's not that impressive yet, but nonetheless you can navigate around and OSM data will be downloaded from Freemap and shown as coloured lines of different width depending on what sort of way it is (see key). Only ways with a highway tag are shown. Use the up and down arrow keys to move forward and back, and the left and right arrow keys to rotate the viewpoint (camera), and P and L to move up and down. Requires a recent Firefox nightly build (Minefield); make sure you update to the latest.

Key

  • White lines are roads; the wider the line, the more important the road.
  • Yellow = footpath
  • Brown = bridleway
  • Red = byway
  • Orange = track
  • Blue = cycleway
